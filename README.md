# AI Chatbot

## Description
This project is a simple AI Chatbot developed using Python. The chatbot interacts with users by responding to text-based queries using predefined rules and basic Natural Language Processing (NLP) concepts.

## Features
- Interactive conversation
- User-friendly interface
- Predefined responses
- Easy to customize
- Beginner-friendly AI project
